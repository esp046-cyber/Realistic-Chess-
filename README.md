# Realistic-Chess-
Mobile-Optimized 3D Chess Game

Features:

Complete chess logic with legal move validation, check/checkmate detection, and turn-based gameplay
Touch-optimized controls with tap-to-select and tap-to-move mechanics
Responsive 3D board using Three.js with proper piece models and smooth animations
Visual feedback including piece highlighting, valid move indicators, and captured piece display
Mobile-first design with portrait orientation support and intuitive gesture controls

Real chess engine with complete rules, castling, en passant, pawn promotion
AI opponent with configurable difficulty (easy/medium/hard)
Advanced move evaluation using piece-square tables and positional analysis
Full 3D visualization with smooth animations and intuitive touch controls
Professional UI with move history, captured pieces, and game status

Features
Complete Chess Rules:

All piece movements (pawn, rook, knight, bishop, queen, king)
Castling (kingside and queenside)
En passant captures
Pawn promotion with piece selection
Check, checkmate, and stalemate detection
Draw detection (50-move rule, insufficient material)

AI Opponent:

Minimax algorithm with alpha-beta pruning for efficient search
Piece-square tables for positional evaluation
Three difficulty levels:

Easy (depth 1) - Good for beginners
Medium (depth 2) - Challenging for casual players
Hard (depth 3) - Advanced strategic play


Evaluates material, position, mobility, and king safety

Game Controls:

Select piece - Tap your white piece to select it
View moves - Green rings show all legal moves
Make move - Tap destination to move
Undo - Take back your last move (undoes both player and AI moves)
New Game - Reset the board
Difficulty - Adjust AI strength in the dropdown

Visual Features:

Smooth piece animations with arcing motion
Captured pieces display
Turn indicator with check warnings
AI thinking indicator
Pawn promotion dialog
Gentle board rotation for 3D perspective
